Syntax Scaffolding Guide

Remember how a count-controlled loop structure looks in Python:

for counter in range(5):
    print("Loop number: " + str(counter))
  • The word counter keeps track of the current number.
  • The str() function must be used to wrap your counter if you want to add it to a sentence!
